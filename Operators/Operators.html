5 + 2 = 7

here, 5 and 2 are the operand and + is a operator.

Unary Operator

It only has single operand.

example: let v = 3;
v++;
// the value of v is now 4;
// v++ is same as v = v + 1;

Binary Operator

If it has two operands

example: let v = 3;
let w = 4;
let x = w + v; // this is binary operator
// the value of x is 7.

Concatenation of String

let s = "1"
alert(1 + s) ; // it gives 11
alert(s + 1); // it also give 11

Number Conversion

alert(+true) // gives 1;
alert(+false) // gives 0;

let s = "3";
let t = "4";
alert(s + t) // gives you 34;

alert(+s + +t) //gives you 7;
OR
alert( Number(s) + Number(t) ); // gives you 7

Remainder

The result of a % b is the remainder of the integer division of a by b.
alert( 4 % 2) // gives you 0

Exponentation

the result of a ** b is a multiplied by itself b times.
alert( 4 ** 2) // gives you 16;
